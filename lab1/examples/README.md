The file 'egfr_egf_genes.fna' in this directory contains nucleotide sequences
downloaded from the US National Center for Biotechnology Information (NCBI)
database.

The sequences correspond to genes encoding the Epidermal Growth Factor (EGF)
and EGF receptor (EGFR) proteins found in humans and mice.

- EGF, Homo sapiens (human): https://www.ncbi.nlm.nih.gov/datasets/gene/1950/
- EGFR, Homo sapiens (humna) https://www.ncbi.nlm.nih.gov/datasets/gene/1956/ 
- EGFR, Mus musculus (mouse): https://www.ncbi.nlm.nih.gov/datasets/gene/13649/

More gene, transcript, protein and genome sequences as well as annotation and
metadata can be found at:

https://www.ncbi.nlm.nih.gov/datasets
